# dotfiles

dotfiles and other customizations...

## Installation

### Linux

The installation requires *GNU stow* (https://www.gnu.org/software/stow/).

```sh
$ mkdir -p ~/.dotfiles && cd ~/.dotfiles
$ git clone https://github.com/aarondettmann/dotfiles.git
$ cd dotfiles
$ ./install.sh
```
